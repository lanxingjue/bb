# RealChanTheory — 真正缠论策略文档

## 概述

完整实现缠中说禅的缠论交易体系：分型→包含处理→笔→中枢→买卖点。
同时融入 MACD 背驰确认 + EMA200 大级别趋势过滤。

## 核心算法

### 1. K线包含处理
- 相邻两根K线有包含关系时合并
- 向上趋势: 取高高（最高价取大值，最低价也取大值）
- 向下趋势: 取低低（最高价取小值，最低价也取小值）

### 2. 分型识别
- 顶分型: 中间K线最高，左右各3根K线都比它低
- 底分型: 中间K线最低，左右各3根K线都比它高

### 3. 笔 (Stroke) 构建
- 底分型→下一个顶分型 = 上升笔 (type=1)
- 顶分型→下一个底分型 = 下跌笔 (type=-1)
- 两分型之间至少间隔 6 根K线

### 4. 中枢 (Pivot) 检测
- 连续 3 笔的价格重叠区 = 中枢
- 上沿 = 3笔最高价的最小值
- 下沿 = 3笔最低价的最大值

### 5. 买卖点定义
- **1买**: 下跌笔结束，价格在中枢下方，EMA50/200多头或底背驰
- **1卖**: 上涨笔结束，价格在中枢上方，EMA50/200空头或顶背驰

### 6. 级别过滤
- 4h级别(EMA200): 只做顺势交易
  - 4h升势(close > EMA200)只做多
  - 4h降势(close < EMA200)只做空
  - 背驰信号可逆势入场

## 策略参数

### 固定参数（策略文件内）
```
timeframe = '1h'
can_short = True
startup_candle_count = 500
use_exit_signal = False
minimal_roi = {"240": 0.015, "120": 0.025, "60": 0.04, "30": 0.06, "0": 0.10}
stoploss = -0.015  (1.5%)
trailing_stop = True
trailing_stop_positive = 0.01
trailing_stop_positive_offset = 0.03
```

### 可调参数（前端面板）
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 仓位占比 | 25% | 每笔 = 本金×25%，动态计算 |
| 单笔止损 | 1.5% | 覆盖策略止损，1.0%~2.0%可调 |
| 杠杆 | 3x | 建议不超过5x |
| 费率 | 0.02(做市)/0.04(吃单) | 做市费率+低滑点效果更好 |
| 滑点 | 0.03~0.05 | 越低越好 |

## 关键发现

### 有效条件
- ✅ **BTC单币** + **做市费率(0.02%)** + **低滑点(0.03%)**
- ✅ 5个月以内中短期回测
- ✅ 4h级别顺趋势交易
- ✅ MACD背驰确认增强信号

### 无效条件
- ❌ 多币种同时交易（摊薄收益）
- ❌ 标准吃单费率(0.04%) + 高滑点(0.05%)
- ❌ 365天全周期（市场风格切换）
- ❌ 逆势交易（4h降势做多、4h升势做空）
