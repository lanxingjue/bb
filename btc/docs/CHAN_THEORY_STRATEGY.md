# RealChanTheory — 真正缠论策略文档

## 概述

完整实现缠中说禅的缠论交易体系：分型→包含处理→笔→中枢→买卖点。
融入 MACD 背驰确认 + EMA50 趋势过滤 + **缠论笔背驰结构出场**（方向四）。

## 核心算法

### 1. K线包含处理
- 相邻两根K线有包含关系时合并
- 向上趋势: 取高高（最高价取大值，最低价也取大值）
- 向下趋势: 取低低（最高价取小值，最低价也取小值）

### 2. 分型识别
- 顶分型: 中间K线最高，左右各3根K线都比它低
- 底分型: 中间K线最低，左右各3根K线都比它高

### 3. 笔 (Stroke) 构建
- 底分型→下一个顶分型 = 上升笔 (type=1)
- 顶分型→下一个底分型 = 下跌笔 (type=-1)
- 两分型之间至少间隔 6 根K线

### 4. 中枢 (Pivot) 检测
- 连续 3 笔的价格重叠区 = 中枢
- 上沿 = 3笔最高价的最小值
- 下沿 = 3笔最低价的最大值

### 5. 买卖点定义
- **1买**: 下跌笔结束，价格在中枢下方，EMA50多头或底背驰
- **1卖**: 上涨笔结束，价格在中枢上方，EMA50空头或顶背驰

### 6. 缠论出场（方向四 — 核心改进）
- 不再完全依赖 trailing_stop 随机出场，加入**笔力度背驰**结构出场
- **背驰出场**：隔笔同向比较，当前笔幅度 < 前同向笔 × 60% → 趋势衰竭 → 出场
- 保留 trailing_stop / minimal_roi 作为风控兜底（优先级低于结构出场）
- 信号出场在 11 个月回测中贡献约 9 笔提前出场，有效锁定利润

### 7. 级别过滤
- EMA50（≈16小时）趋势方向过滤
  - EMA50升势只做多，降势只做空

## 策略参数

### 固定参数（策略文件内）
```
timeframe = '1h'
can_short = True
startup_candle_count = 500
use_exit_signal = False
minimal_roi = {"240": 0.015, "120": 0.025, "60": 0.04, "30": 0.06, "0": 0.10}
stoploss = -0.015  (1.5%)
trailing_stop = True
trailing_stop_positive = 0.01
trailing_stop_positive_offset = 0.03
```

### 可调参数（前端面板）
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 仓位占比 | 25% | 每笔 = 本金×25%，动态计算 |
| 单笔止损 | 1.5% | 覆盖策略止损，1.0%~2.0%可调 |
| 杠杆 | 3x | 建议不超过5x |
| 费率 | 0.02(做市)/0.04(吃单) | 做市费率+低滑点效果更好 |
| 滑点 | 0.03~0.05 | 越低越好 |

## 关键发现

### 有效条件
- ✅ **BTC单币** + **吃单费率(0.04%)** + **滑点(0.05%)** 即可盈利（新突破！）
- ✅ 11个月全周期稳定盈利（旧版无法做到的）
- ✅ 做市费率 + 低滑点表现更强
- ✅ 缠论笔背驰出场显著提升胜率和夏普

### 无效条件
- ❌ 多币种同时交易（摊薄收益，仍不推荐）
- ❌ 逆势交易（降势做多、升势做空）

## 运行方式

### 回测
```bash
# 后端（回测API + 模拟盘API）
cd btc && btc/freqtrade/.venv/bin/python3 backend/main.py
# 浏览器打开 http://localhost:3000
cd btc && npm --prefix frontend run dev
```

### 模拟盘（不需要交易所API Key）
```bash
# 1. 启动后端
cd btc && btc/freqtrade/.venv/bin/python3 backend/main.py

# 2. 启动模拟盘
curl -X POST http://127.0.0.1:8765/api/papertrade/start \
  -H "Content-Type: application/json" \
  -d '{"strategy":"RealChanTheory","pairs":["BTC/USDT:USDT","ETH/USDT:USDT"],"timeframe":"1h","leverage":3}'

# 3. 查看状态
curl http://127.0.0.1:8765/api/papertrade/status

# 4. 停止
curl -X POST http://127.0.0.1:8765/api/papertrade/stop
```

### 数据更新
```bash
# 手动更新一次数据（需要代理）
btc/freqtrade/.venv/bin/python3 scripts/auto_updater.py

# 后台自动更新（每4小时）
btc/freqtrade/.venv/bin/python3 scripts/auto_updater.py --daemon
```
