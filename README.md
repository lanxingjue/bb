# 缠论策略交易系统

基于缠中说禅理论的自动化交易系统，支持回测、模拟盘和实盘交易。

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│  实时数据 (OKX API)                                   │
│  ↓ SOCKS5 代理 (127.0.0.1:1086)                      │
│  realtime_updater (60s) → feather 文件               │
│  ↓                                                   │
│  模拟盘引擎 (实时模式, 非回放)                         │
│  ↓                                                   │
│  FastAPI → 前端 (React + lightweight-charts)         │
└─────────────────────────────────────────────────────┘
```

## 策略

### RealChanTheory (1h 中长线)
- 完整缠论算法：包含处理→分型→笔→中枢→买卖点
- 1买/1卖 + 3买/3卖 + 走势分类 + 背驰出场
- 11个月回测: 60笔, 66.7%胜率, +28.4%, 夏普3.21

### ChanTheoryScalp (5m/1m 超短线)
- 基于 RealChanTheory 核心逻辑，针对短周期优化
- 仅 1买/1卖（中枢外突破，3买已禁用）
- 自适应 EMA/ATR 参数（1m→EMA240/960, 5m→EMA48/192）
- 1m 回测(3个月, 2025): 389笔, 54.5%胜率, +89.9%, 夏普6.85

## 功能

| 功能 | 说明 |
|------|------|
| **回测** | 全周期历史回测，含权益曲线/交易明细/指标面板 |
| **模拟盘** | 实时模式（非回放），动态仓位，K线间止损检查 |
| **诊断** | 按买卖点/币种/出场原因分组分析 (`analyze_signals.py`) |
| **实盘** | 完整风控系统（止损/移动止盈/日亏限额/回撤限额） |
| **数据更新** | SOCKS5 代理自动拉取 OKX 最新 K 线 |

## 快速开始

```bash
# 1. 安装依赖
cd btc
btc/freqtrade/.venv/bin/pip install -r requirements.txt

# 2. 启动后端
btc/freqtrade/.venv/bin/python3 btc/backend/main.py

# 3. 启动前端 (新开终端)
npm --prefix btc/frontend run dev

# 4. 启动实时数据更新器 (新开终端)
btc/freqtrade/.venv/bin/python3 btc/scripts/realtime_updater.py

# 5. 启动模拟盘
curl -X POST http://127.0.0.1:8765/api/papertrade/start \
  -H "Content-Type: application/json" \
  -d '{"strategy":"ChanTheoryScalp","pairs":["BTC/USDT:USDT"],"timeframe":"1m","leverage":3}'

# 6. 打开浏览器
open http://localhost:3000
```

## 关键技术决策

| 决策 | 说明 |
|------|------|
| **同步时间轴** | 多币种模拟盘按统一时间戳推进，防止前视偏差 |
| **实时模式** | 模拟盘从最新数据开始，等待新K线，非历史回放 |
| **K线间止损** | 每10秒用最新价检查止损，不等新K线收盘 |
| **状态持久化** | 交易/持仓保存到 JSON，重启不丢 |
| **出场顺序** | 止损 > 移动止盈 > 结构出场 > ROI |
| **动态仓位** | 持仓越多单笔越小，总暴露不超过50% |
| **SOCKS5 代理** | HTTP代理被交易所屏蔽，SOCKS5 正常工作 |
| **数据缺口补全** | 15m插值生成1m，覆盖无交易所原始1m的时段 |
